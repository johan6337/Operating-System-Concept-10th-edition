# Question 1.19 #

Rank the following storage systems from slowest to fastest:

- a. Hard-disk drives
- b. Registers
- c. Optical disk
- d. Main memory
- e. Nonvolatile memory
- f. Magnetic tapes
- g. Cache

# Answer #

The ranking of the storage systems from fastest to slowest is as follows:

- b. Registers
- g. Cache
- d. Main memory
- e. Nonvolatile memory
- c. Optical disk
- a. Hard-disk drives
- f. Magnetic tapes
